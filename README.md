# AChE – Donepezil Docking Project

## Overview
Molecular docking study of donepezil with acetylcholinesterase (AChE, PDB ID: 1EVE)
performed using AutoDock Vina.

## Tools
- AutoDock Vina
- PyMOL
- Meeko
- Python (conda environment)

## Docking Setup
Active site: Ser203, Glu334, His447  
Grid center: (6.503, 68.673, 63.189)  
Grid size: 22 × 22 × 22 Å

## Results
Best binding affinity ≈ −10 kcal/mol.
